Sobre o sistema

Esse é um sistema de aluguel/empréstimo de livros de uma biblioteca, no qual é possível cadastrar um livro e um cliente.

Para um cliente emprestar um livro, deve obrigatoriamente registrado no sistema. Na tela de aluguel, é possível consultar os livros e os clientes registrados, e através do código do livro e código do cliente fazer um empréstimo. Quando emprestar um livro, o sistema irá gerar uma data para devolução na qual se não devolvido até a data correspondente, o cliente irá ser cobrado em uma multa no valor de R$15,00 por livro emprestado.
